Overview:
This project is an intelligent navigation system designed to assist visually impaired individuals in navigating their surroundings safely and independently. The system uses a Convolutional Neural Network (CNN) and Computer Vision techniques to detect obstacles in real-time and provide audio guidance to the user.

Features :
Real-time obstacle detection
CNN-based image classification
Voice feedback for navigation assistance
User-friendly interface using Streamlit
Camera-based environmental monitoring
Improved mobility and safety for visually impaired users
Technologies Used
Python
TensorFlow / Keras
CNN (Convolutional Neural Network)
OpenCV
Streamlit
NumPy
Pandas
Working
Capture real-time images using a webcam.
Preprocess the captured images.
Feed images into the trained CNN model.
Detect and classify obstacles.
Generate voice alerts and navigation instructions.
Assist users in avoiding obstacles and moving safely.
Project Objectives
Enhance independent mobility for visually impaired individuals.
Detect obstacles accurately in real time.
Provide immediate audio-based navigation support.
Utilize AI and Deep Learning for assistive technology.
Future Enhancements
GPS-based navigation support.
Integration with wearable devices.
Advanced object detection using YOLOv8.
Multilingual voice assistance.
Outdoor and indoor navigation optimization.
Applications
Smart assistive devices
Healthcare and accessibility solutions
Indoor navigation systems
Public space assistance
